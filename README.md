Hubs, Multiple Conditions, and Complex Design.

Installation

### devtools ###
R packages vegan, plyr installed.

From an interactive R session:

```{r, eval=FALSE}
library(devtools)
install_github("jennylsl/MicroBNetwork")
library(MicroBNetwork)
```
